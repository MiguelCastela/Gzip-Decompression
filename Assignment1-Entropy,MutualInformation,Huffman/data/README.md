# data

This folder must contain `CarDataset.xlsx`, the dataset analysed by the
project. It is the course-provided car dataset and is not redistributed here.

Place the file as:

```
data/CarDataset.xlsx
```

The dataset has seven numeric columns, in this order:

| Index | Column       | Meaning                              |
|-------|--------------|--------------------------------------|
| 0     | Acceleration | Time to reach a set speed            |
| 1     | Cylinders    | Number of engine cylinders           |
| 2     | Displacement | Engine displacement                  |
| 3     | Horsepower   | Engine horsepower                    |
| 4     | ModelYear    | Year of the model                    |
| 5     | Weight       | Vehicle weight                       |
| 6     | MPG          | Miles per gallon (the target value)  |
